# Educational NGO Management System

A web-based management system for educational NGOs built with .NET 8 and MySQL.

## Features

- Beneficiary Management
- Volunteer Management
- Donation Management
- Event Management
- Mobile Responsive Design

## Prerequisites

- Visual Studio 2022
- .NET 8 SDK
- MySQL Server
- Git
